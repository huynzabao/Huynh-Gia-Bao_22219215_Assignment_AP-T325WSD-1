**Project: Operational Risk Analytics & Data Cleansing Pipeline**

**1. Executive Summary**

This project demonstrates the application of data analytics to identify operational bottlenecks and risks. By processing complex diagnostic logs, the objective was to transform raw, messy data into structured, actionable business insights for risk management.

**2. Business Value & Financial Transferability**

While built in R, the logic of this data pipeline directly translates to modern financial operations (such as using Excel Power Query and Data Modeling). The project showcases core competencies in:

- **Data Cleansing:** Handling missing values securely (Median Imputation)  to ensure data integrity before reporting.

- **Risk Identification:** Pinpointing the Top 5 critical operational risks  from unstructured logs.

- **Performance Validation:** Using statistical hypothesis testing (Welch t-tests)  to rigorously prove that observed variances are statistically significant, rather than just random market noise.

**3. Tools & Methodologies**

- **Language:** R (tidyverse)

- **Version Control:** GitHub (Demonstrating structured workflow and reproducible reporting) 

- **Techniques:** Median Imputation, Exploratory Data Analysis (EDA), Welch t-tests.

**4. Key Findings**

- Successfully engineered a robust pipeline that automates the cleaning of diagnostic logs.

- Identified and ranked the Top 5 critical operational risks, providing a data-backed foundation for management to allocate resources for risk mitigation.
